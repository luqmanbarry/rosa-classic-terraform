# Playbooks

This directory contains Ansible Automation Platform examples for running the factory from automation.

- `aap/run_cluster_workflow.yml`: run validate, plan, or apply for one cluster by calling the shared factory workflow script.
- `aap/README.md`: AAP-specific inputs, prerequisites, and example usage.

Use `docs/operations/aap-execution.example.yml` as the starting point for AAP extra vars.
