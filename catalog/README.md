# Catalog

This directory stores shared defaults used by cluster stacks.

- `cluster-classes/`: environment or platform defaults shared across multiple clusters
- `machine-pool-classes/`: reusable worker pool defaults

Cluster class examples now include:

- `dev.yaml`
- `qa.yaml`
- `prod.yaml`
- `prod-dr.yaml`

Machine pool class examples now include:

- `system.yaml`
- `observability.yaml`
- `aap.yaml`
- `ai.yaml`
